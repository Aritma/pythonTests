### QuestionMaker5000
Just for fun, because I need graphical tool to create ABCD tests.
Supports UTF-8 (tested with CZ)

##**USAGE**

`python quizapp.py questions.json`

Can use any json with similar data.

##**REQUIREMENTS**

Python 3
